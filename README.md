# Ritsumeikan-Dijkstra

立命館大学の広大な土地で迷わないよう，さらに授業間の教室移動が間に合うように，建物間の最短経路を求めるプログラムを書きました．
プログラムの手順は

1. マップ上の重要なポイントに赤いマークをして，
2. その座標を取得し，
3. 座標を圧縮し，
4. 座標間の距離を求め，
5. ダイクストラ法で実行する

と言ったものになっています．なお，まだ完成していません．
